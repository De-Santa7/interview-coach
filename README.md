# InterviewIQ — AI Interview Coach

Practice job interviews that actually prepare you. InterviewIQ generates 
real interview questions for any profession, lets you answer by typing or 
voice, gives you a role-specific practical challenge, then delivers a full 
AI-powered hiring verdict at the end.

## Features
- 50+ professions supported — tech, healthcare, law, engineering, education and more
- Voice answers via Web Speech API — no extra setup needed
- Role-specific practical challenges — code editor for devs, case studies for doctors, campaign briefs for marketers
- Full session report — overall score, per-question breakdown, ideal answers, and a hiring verdict
- Google sign-in via Supabase Auth
- Streaming AI responses powered by Groq (llama-3.3-70b-versatile)

## Tech Stack
Next.js · TypeScript · Tailwind CSS · Supabase · Groq API
